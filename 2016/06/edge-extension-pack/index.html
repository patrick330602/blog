Edge Extension Package released

Jun 20, 2016 in Open Source Life

Recently I realize it is still a pain to find extensions for edge, but looking around GitHub and Stackflow, I actually cannot find anything that can easily install(or you can call register ) unoffical Edge extensions. So, I made a windows batch file for easy deployment. Introducing the new Edge Extensions Pack.

Working Extensions

This is a pack of unoffical Edge extensions pack for easy install that currently included:

  • Breeze - New Tab
  • CatBlock For AdBlock
  • Flying YouTube
  • Instant Translate
  • Turn Off the Lights
  • Tweeten
  • Legacy Offical Extension(Not install by default)
    • Microsoft Translator
    • Mouse Gestures
    • OneNote Clipper
    • Pin It Button
    • Reddit Enhancement Suit (RES)

It is very easy to use:

  1. Dowload package or git clone https://github.com/patrick330602/Edge-Extensions-Pack.git
  2. Click Setup.cmd
  3. Use the method on the Edge Extension Developer Website to add extensions
  4. Done!

Also, if you don’t mind, you can access the command prompt for advance use.

Command Usage
setup Normal install
setup -l setup legacy Installing with legacy extensions
